# glut-vue-demo
GLUT应用模版和实例

## 克隆项目到本地

```bash
git clone git@github.com:LeeLejia/glut-vue-demo.git
```

## 安装依赖

```javascript
npm i
```

## 调试

```bash
npm run dev
// 在glut调试面板输入链接：http://localhost:5656/build.js
```

## 发布

```bash
npm run build
// 在发布面板将打包后的文件/dist/prod-build.js,发布为glut应用
```
